# Projeto: Dashboard de Anúncios de Carros

Este projeto é um aplicativo web interativo desenvolvido em Python utilizando Streamlit para análise exploratória de dados de anúncios de carros usados nos Estados Unidos.

## Funcionalidades

- Visualização de dados do dataset
- Histograma da quilometragem dos veículos
- Gráfico de dispersão entre preço e quilometragem
- Filtro por tipo de veículo

## Aplicação online

O aplicativo pode ser acessado no seguinte link:

https://projeto-carros-khjt.onrender.com/

## Tecnologias utilizadas

- Python
- Pandas
- Plotly Express
- Streamlit
