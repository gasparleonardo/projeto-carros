# Análise de anúncios de carros

Este projeto é um aplicativo web desenvolvido com Streamlit para analisar
dados de anúncios de carros usados nos Estados Unidos.

O aplicativo permite:

- visualizar um histograma da quilometragem dos carros
- visualizar um gráfico de dispersão entre quilometragem e preço

As visualizações são criadas usando Plotly Express.
